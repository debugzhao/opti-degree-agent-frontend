# 志愿参谋 Agent React MVP

高考志愿填报 Agent 产品原型，使用 React + Vite 实现。

## 功能

- 首页直接开始测算：省份、分数、位次、选科、截图上传演示
- 免费风险画像：数据完整度、省份覆盖、调剂/专业组风险
- 家庭建档：家庭背景、预算、城市偏好、专业偏好、不可接受专业
- 三套方案：保守型、均衡型、进取型冲稳保推荐
- 付费交付：¥99、¥399、¥999、¥1999 套餐
- 志愿表风险体检：保底不足、热门扎堆、梯度过密、不可接受专业
- 顾问复核：AI 咨询底稿、复核清单、家庭会议议程
- 家庭协同：学生/父亲/母亲分别标注喜欢、不能接受、有疑问

## 运行

```bash
npm install
npm run dev
```

当前版本使用 `src/data.js` 的模拟数据。上线前需要替换为官方招生计划、历年投档线、一分一段、专业选科要求、体检限制、学费、招生章程和学校就业质量报告。
